Area 3

Library — the real bookshelf

Covers-first, always (the #1 gripe with Libib). Personal shelves are chips, a book’s page is split into the book (shared catalog) and your copy (private), and adding a book is a camera-first flow.

Cover treatments. Every cover — generated or real — sits in the same frame: rounded corners, left spine shade, drop shadow. With no image, Kitabi typesets a quiet cover from title + author on a colour derived from the book. When the user uploads a photo (or the catalog has one), the image fills the same frame edge-to-edge, and favourites ribbons or lending bands overlay it identically — so a shelf of mixed real and generated covers still reads as one bookshelf.
